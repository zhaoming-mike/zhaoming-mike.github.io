# Changelog

## 2026-06-12

- 新增荣誉排行页面：标兵榜（最近60日DPG变化最大的前10人）、劳模榜（最近60日场次最多的前10人）
- 每人独立卡片展示，3列网格布局，手机端自动收为单列
- 卡片信息：排名（前三🥇🥈🥉奖牌）、昵称、军衔图标、WN8/WNX/DPG（区分60日和总战绩）、数值着色使用 Wotlabs 色阶

## 2026-05-22

- 新增组队关系网络：D3.js 力导向图可视化军团成员间的组队关系
- 组队数据采集脚本：单线程顺序抓取 tomato.gg Advanced 页 Teammates 卡片，逐条即时写入 JSON，支持断点续传
- 只采集近 60 天内有战斗记录的活跃成员，按场次降序优先采集
- 节点颜色按场均伤害着色（紫≥1500 → 红<300），节点大小按组队人数成正比
- 连线粗细代表一起组队场次（7 级），连线颜色代表组队胜率
- 连线标注场次数字（≥10 场显示）
- 节点文字显示在圆形正中间，节点直径与连线数量成正比
- 点击节点高亮其相连节点和连线，其余元素变灰；再次点击或点空白取消
- 节点排斥力滑块（-100 ~ -2000），节点大小滑块（0.5x ~ 3x），拖动即时生效
- 边界排斥力防止节点超出画布，滚轮缩放裁剪平移范围避免内容跑出视野
- 最少场次过滤下拉框（1/3/5/10/20/50 场）
- 刷新按钮手动重新加载不断更新的组队数据
- 卡片头部图例说明：节点颜色、节点大小、连线粗细、连线颜色、点击交互

## 2026-05-21

- 成员列表新增对比功能：勾选 2 名成员后点击「对比选中」按钮，弹窗并排展示全部统计和 60 日数据差异，数值高者绿低者红标注

## 2026-05-20

- 所有展示 WN8 的位置同步加入 WNX 数据（60日排名表、成员详情页）
- PR 排行卡片重命名为「个等排行 (Personal Rating)」，场均排行重命名为「场均排行 (DPG = Damage Per Game)」
- 60日排名新增多条件过滤：DPG、WN8、WNX、胜率区间、存活率、KD，支持且/或逻辑切换
- 个等排行与场均排行卡片右上角增加「更多」按钮，弹窗展示全员完整排行
- 全员排行弹窗支持搜索昵称、角色筛选、场次下限过滤，移除滚动条改为页面自然滚动
- 首页新增更新日志卡片，位于数据同步旁，点击弹窗展示完整 changelog
- 图表颜色全部改用 tomato.gg Wotlabs 10级色阶，与全局数值着色保持一致
- 胜率分布图按区间分级着色（<44% 红 → 60%+ 蓝青）
- 散点图每个点按对应成员统计值着色，悬停显示昵称和具体数值
