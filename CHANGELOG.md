# Changelog

## 2026-05-21

- 成员列表新增对比功能：勾选 2 名成员后点击「对比选中」按钮，弹窗并排展示全部统计和 60 日数据差异，数值高者绿低者红标注

## 2026-05-20

- 所有展示 WN8 的位置同步加入 WNX 数据（60日排名表、成员详情页）
- PR 排行卡片重命名为「个等排行 (Personal Rating)」，场均排行重命名为「场均排行 (DPG = Damage Per Game)」
- 60日排名新增多条件过滤：DPG、WN8、WNX、胜率区间、存活率、KD，支持且/或逻辑切换
- 个等排行与场均排行卡片右上角增加「更多」按钮，弹窗展示全员完整排行
- 全员排行弹窗支持搜索昵称、角色筛选、场次下限过滤，移除滚动条改为页面自然滚动
- 首页新增更新日志卡片，位于数据同步旁，点击弹窗展示完整 changelog
- 图表颜色全部改用 tomato.gg Wotlabs 10级色阶，与全局数值着色保持一致
- 胜率分布图按区间分级着色（<44% 红 → 60%+ 蓝青）
- 散点图每个点按对应成员统计值着色，悬停显示昵称和具体数值

## 2026-05-19

- 引入 tomato.gg Wotlabs 10级色阶，全局统一数值着色（WN8/WNX/胜率/DPG/存活率/KD/场均击杀）
- color-scale.js 独立模块，配置对象 Object.freeze 锁定，页面加载后不可篡改
- 响应式布局适配手机/平板/桌面三端
- 手机端保存图片调用 Web Share API / canvas 导出到相册
- 首页拆分为军团简介与数据同步两个独立卡片
- 卡片间距统一 16px，统计卡片改为水平行内格式
- 军衔图标调大至 28px
- 所有无颜色定义的数值默认白色
- Rating 统一显示为 PR
- 战斗官 → 指战官，成员 → 士兵
- 排行榜两列展示 10 人，移除滚动条
- 数据采集脚本增加 `_generated_at` 时间戳
